# Pasta de Imagens de Produtos

Coloque as imagens dos produtos nesta pasta.

## Como nomear as imagens:

Use nomes descritivos e sem espaços, por exemplo:
- `anel-diamante-ouro.jpg`
- `colar-pérola-prata.jpg`
- `brinco-prata-925.png`

## Formato recomendado:
- **Extensões**: `.jpg`, `.jpeg`, `.png` ou `.webp`
- **Tamanho**: Recomendado 800x800px ou maior (quadrado)
- **Peso**: Otimize as imagens para web (máximo 500KB por imagem)
