# Imagem do Hero

Coloque aqui a imagem principal do banner/hero da página inicial.

## Recomendações:
- **Tamanho**: 1200x800px ou maior
- **Formato**: JPG, PNG ou WebP
- **Peso**: Máximo 500KB (otimizado para web)
- **Nome sugerido**: `hero-banner.jpg` ou `hero-main.jpg`

A imagem será exibida no topo da página inicial como banner principal.
